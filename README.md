### Build Setup

#### Client
```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test

```
#### Server
```
# install dependencies
npm install

# conecction to MONGODB
npm run dev
```
